Weather Dashboard

For the Frontend I..

Built the UI using React functional components.

Managed application and UI state using useState (locations, selected location, weather data, UI visibility).

Used useEffect to: Fetch saved locations on initial render. Fetch weather data when the active location changes.

Rendered hourly and daily forecasts using dynamic list rendering.

Implemented horizontal scrolling layouts using CSS Flexbox for forecast cards.

Displayed weather conditions using dynamically selected icons based on weather codes.

For the Backend I..

Implemented a RESTful API using FastAPI.

Used SQLAlchemy to persist saved locations in a database.

Exposed endpoints to: Add locations, Delete locations, Retrieve stored locations, Fetch weather data for a selected location

I created this full-stack weather dashboard using React with hooks for state management and lifecycle handling. The frontend consumes a RESTful FastAPI backend backed by SQLAlchemy for persistent location storage, and displays current, hourly, and daily forecasts using data from the Open-Meteo API.

The application uses a dedicated data access layer written in C# to interact with MySQL, including stored procedures and transactional queries for account management and book lending operations. CLI File Encryption Program

This is my simple aes file encryption CLI app. I wrote it in Rust and it works pretty well!

It securely encrypts and decrypts files using a password-derived key, a random salt, and a nonce. The tool is cross-platform, uses aes-gcm and rand crates. I used it for learning symmetric encryption and secure key derivation in Rust.
